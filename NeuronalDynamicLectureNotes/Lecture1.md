# 1.2 passive membrane

### R-C circuit

$$
I(t): current\ injected\ at\ time\ t\\
U_{rest}: rest\ voltage\\
I(t) = I_R(t)+I_C(t)\\
I_C(t) = C\cdot \frac{dU}{dt}\\
I_R(t) = \frac{1}{R}(U-U_{rest})\\
R\cdot C\cdot\frac{dU}{dt} = -(U-U_{rest}) + R\cdot I(t)\\
\tau\cdot\frac{dU}{dt} = -(U-U_{rest}) + R\cdot I(t)\\
\tau\cdot\frac{dV}{dt} = -V + R\cdot I(t)\\
$$
